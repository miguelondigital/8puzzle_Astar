# 8puzzle_Astar
Python 8 puzzle solver using A* 

Uses 2 distinct heurisitcs
- **H1** - Number of misplaced tiles
- **H2** - Manhattan distance

# How to Run
```bash
python3 proj1.py
```

# Menu Options
1. Random Board - Generates random solveable board and solves it with user chosen heuristic
2. Manual Input - Allows user to input custom board (**See instructions below**)
3. 100 Case Statistical Test - Runs 100 cases with both heuristics for comparison
4. Exit

# Manual Input Cont.
Once you have input the board which is one row per line and space seperated press **Enter**. Then
- **Ctrl+D** for Linux/macOS
- **Ctrl+Z then Enter** for windows
- **Cmd+D** macOS alternative
This signals the end of the input to allow the program to continue and read the board.
